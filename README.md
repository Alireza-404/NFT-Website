# 🎨 NFT-Website

A modern and **visually appealing NFT landing page** built with **HTML**, **CSS**, and **Bootstrap**.  
This project focuses solely on design, aesthetics, and responsiveness — no dynamic functionality or backend logic is implemented.

---

## 🌟 Overview

NFT-Website demonstrates a **sleek, modern, dark-themed design** for showcasing NFT content.  
It is **fully responsive**, designed to look great on both desktop and mobile devices.  
The goal of this project is to highlight **UI design skills** and **Bootstrap layout techniques**.

> This project is purely for **visual presentation** and does not include any functional logic or interactivity beyond standard HTML/CSS effects.

---

## 🧠 Key Features

- 🖤 **Dark-themed modern design**
- 📱 **Fully responsive** layout
- 🎨 **Clean and aesthetic UI**
- 🏗️ **Bootstrap grid system** for layout
- 🔤 **Custom fonts** included
- 🖼️ **Image assets and icons** for NFT showcase

## 🧩 How It Works

- This is a **static single-page website**.
- Layout and responsiveness are handled using **Bootstrap** classes.
- Design elements like **colors, fonts, and spacing** are managed via CSS.
- All assets (images, fonts, icons) are stored locally in their respective folders.
